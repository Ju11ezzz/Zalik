# 🗒️ Color Notes — застосунок для роботи з нотатками

**Color Notes** — це мобільний застосунок, створений на **React Native** з використанням **Expo**. Проєкт виконано як залікову роботу з дисципліни **«Розробка інтерфейсів користувача»**.

Застосунок призначений для створення та зберігання особистих нотаток. Користувач може додавати нові записи, змінювати вже створені, видаляти непотрібні нотатки, шукати записи за текстом і розділяти їх за кольоровими категоріями.

## 🔹 Можливості застосунку

- **Додавання нотаток** — створення нового запису із заголовком, основним текстом і вибором кольору
- **Редагування записів** — можливість змінити назву, текст або колір уже створеної нотатки
- **Видалення нотаток** — видалення запису з додатковим підтвердженням дії
- **Пошук по нотатках** — пошук виконується за заголовком і вмістом нотатки
- **Кольорові категорії** — кожна нотатка може мати власний колір для зручного візуального поділу
- **Фільтрація за кольором** — відображення нотаток тільки вибраної кольорової категорії
- **Локальне збереження** — дані зберігаються на пристрої та не зникають після перезапуску додатку
- **Обробка порожніх станів** — окремі повідомлення для випадків, коли нотаток ще немає або пошук не дав результатів

## ⚙️ Використані технології

- **React Native** — основа для створення мобільного інтерфейсу
- **Expo** — середовище для запуску, тестування та розробки застосунку
- **JavaScript** — мова програмування проєкту
- **React Navigation** — перехід між головним екраном і редактором нотатки
- **AsyncStorage** — збереження нотаток у локальному сховищі
- **React Hooks** — робота зі станом, пошуком, фільтрацією та оновленням екранів

Короткий опис основних частин проєкту:

- `App.js` — головний файл застосунку, у якому підключено навігацію між екранами
- `src/screens/` — екрани додатку: список нотаток і форма створення/редагування
- `src/components/` — окремі компоненти інтерфейсу, які використовуються на екранах
- `src/constants/` — файл із кольорами інтерфейсу та кольорами для нотаток
- `src/storage/` — логіка збереження, читання, оновлення та видалення нотаток

## 🧭 Інструкція користувача

### Створення нової нотатки

1. На головному екрані натиснути кнопку створення нотатки.
2. Ввести заголовок.
3. Додати основний текст.
4. Вибрати колір нотатки.
5. Натиснути кнопку **Зберегти**.

### Редагування нотатки

1. Вибрати потрібну нотатку зі списку.
2. Змінити заголовок, текст або колір.
3. Натиснути **Зберегти зміни**.

### Видалення нотатки

1. Відкрити потрібну нотатку.
2. Натиснути **Видалити нотатку**.
3. Підтвердити видалення у діалоговому вікні.

### Пошук та фільтрація

- У полі пошуку можна ввести частину заголовка або тексту нотатки.
- Для перегляду нотаток певної категорії потрібно вибрати відповідний колір.
- Кнопка **Усі** скидає фільтр за кольором.
- Якщо за пошуком або фільтром нічого не знайдено, можна скористатися кнопкою **Скинути фільтри**.

## 🚀 Запуск проєкту

Клонувати репозиторій:

```bash
git clone https://github.com/Ju11ezzz/Zalik.git
```

Перейти в папку проєкту:

```bash
cd ColorNotes
```

Встановити залежності:

```bash
npm install
```

Запустити застосунок:

```bash
npx expo start
```

Після запуску можна:

- відкрити застосунок через **Expo Go** на телефоні;
- відсканувати QR-код із терміналу;
- або запустити проєкт на Android Emulator.
