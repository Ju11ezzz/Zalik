Color Notes
Мобільний додаток-нотатник, розроблений на React Native з використанням Expo.

Тема роботи
Залікова робота з дисципліни «Розробка інтерфейсів користувача».

Варіант №16
Реалізувати додаток-нотатник з можливістю створення, редагування та видалення нотаток. Додати пошук за текстом нотатки та категоризацію за кольором.

Запланований функціонал
створення нотаток;
редагування нотаток;
видалення нотаток;
пошук за текстом нотатки;
категоризація нотаток за кольором;
локальне збереження даних.

Технології
React Native
Expo
JavaScript

Запуск проєкту
Встановити залежності:

npm install


Запустити додаток:

npx expo start


Відкрити додаток через Expo Go або Android Emulator.
